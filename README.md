# Online Shop AR - SMM Panel Project

এটি একটি উন্নত সোশ্যাল মিডিয়া মার্কেটিং প্যানেল। 

## ফিচারসমূহ:
- ✅ অটোমেটেড পেমেন্ট সিস্টেম (বিকাশ/নগদ)
- ✅ সুরক্ষিত লগইন এবং সাইন-আপ সিস্টেম
- ✅ উন্নত অ্যাডমিন প্যানেল
- ✅ ক্লাউডফ্লেয়ার সিকিউরিটি ইন্টিগ্রেশন

## কিভাবে রান করবেন:
1. আপনার হোস্টিংয়ে ডাটাবেস তৈরি করুন।
2. `config.php` ফাইলে ডাটাবেস তথ্য দিন।
3. `public_html` ফাইলে সব কোড আপলোড করুন।
